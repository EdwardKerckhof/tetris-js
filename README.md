# Tetris js

Tetris game made with Typescript and canvas. This in a simple Vue 3 frontend.

### Install

```bash
git clone https://github.com/EdwardKerckhofZoneDev/tetris-js.git
cd tetris-js
yarn install
yarn dev
```
